六级标题

用#来表示标题，#数量代表着几级标题，最高是六级也就是六个＃。
可以用-空格的方式来表示无序项目列表。

  1. 项目1
  2. 项目2
  3. 项目3
    有序项目列表也可以用数字+英文句点的方式表示。
    要在保留列表连续性的同时在列表中添加另一种元素，请将该元素缩进四个空格或一个制表符。
    例如：

  • This is the first list item.

  • Here’s the second list item.

    I need to add another paragraph below the second list item.

  • And here’s the third list item.
    或者你也可以使用引用

  • This is the first list item.

  • Here’s the second list item.

    A blockquote would look great below the second list item.

  • And here’s the third list item.
    链接文字
    链接可以用的方式表示，[]里面的文字是链接的文字，()里面要写链接地址。

    图片可以用的方式表示，![]要写图片描述，()里面要写图片地址。
    斜体
    加粗
    斜体可以用*，加粗可以用**，如果要表示斜体加粗的文字，可以用*** *。
    代码块可以用三个单或双引号+代码+三个单或双引号表示，代码块可以包含多行代码。
    行内代码
    代码块的行内代码可以用反引号`表示。
    分割线可以用三个星号    或三个减号—表示。

语法高亮可以用三个单引号+python表示Python代码块，也可以用三个单引号+javascript表示JavaScript代码块。
删除文本
双波浪线表示删除线

注释可以用（!-- --）来表示，记得把（）换成<>。

这是一个引用
记得用右尖角，可以用多个右尖号表示嵌套引用。

这是一个分割线用多个减号组成的。
表格可以用|来分割单元格，-来分割表头和内容，:来对齐内容。

表头1表头2
内容1内容2
